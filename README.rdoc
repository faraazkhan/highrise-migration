== Highrise Migration Tool {<img src="https://travis-ci.org/faraazkhan/highrise-migration.png" />}[https://travis-ci.org/faraazkhan/highrise-migration]



Highrise Migration Tool is a simple Rails application that can be used to transfer data from one Highrise Account
into another Highrise Account. 

Supported Data includes: Person, Companies, Deal, DealCategory, Kase, Note, Task, TaskCategory
